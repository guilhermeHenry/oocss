# oocss

prefixo + (-) + valor da propriedade.

```css
 .bg-red{background: red;}
 ```

| prefixo | name             | values exemple              |
| ------- | :--------------- | --------------------------- |
| bg      | background-color | red, green, yellow, blue... |
| fl      | float            | left, right, insue          |
| tx      | text             |                             |
